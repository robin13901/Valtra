```dart
PreferredSizeWidget buildGlassAppBar({
  required BuildContext context,
  required String title,
  List<Widget>? actions,
  Widget? leading,
  bool centerTitle = true,
});

Widget buildGlassFAB({
  required BuildContext context,
  required IconData icon,
  required VoidCallback onPressed,
  String? tooltip,
});

class GlassCard extends StatelessWidget {
  final Widget child;
  final EdgeInsetsGeometry? padding;
  final EdgeInsetsGeometry? margin;
  const GlassCard({required this.child, this.padding, this.margin});
}
```

Note: buildGlassFAB wraps FloatingActionButton in a Container. Test finders using
`find.byType(FloatingActionButton)` will still work because the FAB is inside the Container.
buildGlassAppBar does NOT support `bottom:` parameter. For smart_plug_consumption_screen
which uses AppBar.bottom for room subtitle, either move subtitle into the title or pass it
as a subtitle in the title text.
