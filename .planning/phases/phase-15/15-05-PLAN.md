# Plan 15-05: Heating Meter Rework — UI Layer

**Requirement**: FR-13.3 (Heating Meter Rework)
**UAC**: UAC-M3-13
**Wave**: 2b (Depends on Plan 15-04 data layer)

## Goal
Rework heating meter UI: mandatory room selection, remove location field, group meters by room, heating type selector with ratio input.

## Current State (after Plan 15-04)
- Data layer updated: roomId FK, heatingType, heatingRatio
- HeatingScreen still shows flat list with optional location subtitle
- HeatingMeterFormDialog still has location text field

## Tasks

### T1: Rework HeatingMeterFormDialog
**Files**: `lib/widgets/dialogs/heating_meter_form_dialog.dart`
1. Remove location text field
2. Add room selection dropdown (required, same pattern as smart plug form)
3. Add heating type selector: "Eigener Zähler" / "Zentralheizung" (SegmentedButton or Dropdown)
4. When "Zentralheizung" selected: show ratio input field (percentage, 1-100, converts to 0.01-1.0)
5. When "Eigener Zähler" selected: hide ratio field
6. Return `HeatingMeterFormData` with `name`, `roomId`, `heatingType`, `heatingRatio`

### T2: Rework HeatingScreen — group by room
**Files**: `lib/screens/heating_screen.dart`
1. Group heating meters by room (same pattern as SmartPlugsScreen)
2. Each room section: room name header + list of meters in that room
3. Each meter card: name, latest reading, heating type indicator
4. For central_meter: show ratio badge (e.g., "25%")
5. Remove location display (replaced by room grouping)
6. Keep expandable readings list per meter
7. FAB: add new meter (requires at least one room — show message if no rooms)
8. Room icon in app bar: navigate to RoomsScreen

### T3: Localize new strings
**Files**: `lib/l10n/app_en.arb`, `lib/l10n/app_de.arb`
1. Add: `heatingType` ("Heating type" / "Heizungstyp")
2. Add: `ownMeter` ("Own meter" / "Eigener Zähler")
3. Add: `centralHeating` ("Central heating" / "Zentralheizung")
4. Add: `heatingRatio` ("Heating ratio (%)" / "Heizanteil (%)")
5. Add: `heatingRatioHint` ("Share of total heating energy" / "Anteil an der Gesamtheizenergie")
6. Remove: location/standort keys (if only used in heating meter form)

### T4: Tests
**Files**: `test/widgets/dialogs/heating_meter_form_dialog_test.dart`, `test/screens/heating_screen_test.dart`
1. Form tests: room dropdown required, heating type selector toggles ratio visibility, ratio validation (1-100)
2. Screen tests: room grouping display, central meter ratio badge, FAB behavior with no rooms
3. Verify no location field remains in form or screen
