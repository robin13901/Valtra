# Plan 15-03: Smart Plug Entry Rework — UI Layer

**Requirement**: FR-13.2 (Smart Plug Entry Rework)
**UAC**: UAC-M3-12
**Wave**: 2b (Depends on Plan 15-02 data layer)

## Goal
Rework smart plug consumption form and screen: month/year picker replaces interval type + start date.

## Current State (after Plan 15-02)
- Data layer updated: `month` column, no intervalType
- Form still has old UI fields
- Screen still displays interval type labels

## Tasks

### T1: Rework SmartPlugConsumptionFormDialog
**Files**: `lib/widgets/dialogs/smart_plug_consumption_form_dialog.dart`
1. Remove interval type dropdown
2. Remove date picker (start date)
3. Add month/year picker:
   - Two dropdowns: month (Jan-Dec, localized) + year (2020-current+1)
   - Default: current month/year for new entries
   - Display as "März 2026" / "March 2026" (localized)
4. Keep kWh value field (unchanged)
5. Update `SmartPlugConsumptionFormData`: `month` (DateTime, 1st of month) + `valueKwh`
6. Validation: warn if entry exists for selected month (show inline message, allow overwrite)

### T2: Update SmartPlugConsumptionScreen
**Files**: `lib/screens/smart_plug_consumption_screen.dart`
1. Update list item display: "März 2026 — 42.5 kWh" format
2. Remove interval type label
3. Keep edit/delete functionality with updated data model

### T3: Localize new strings
**Files**: `lib/l10n/app_en.arb`, `lib/l10n/app_de.arb`
1. Add: `selectMonth` ("Select month" / "Monat wählen")
2. Add: `consumptionForMonth` ("Consumption for {month}" / "Verbrauch für {month}")
3. Add: `entryExistsForMonth` ("Entry already exists for this month. It will be updated." / "Eintrag für diesen Monat existiert bereits. Er wird aktualisiert.")
4. Remove unused: `intervalType`, `intervalStart`, `daily`, `weekly`, `yearly` (only if not used elsewhere)

### T4: Tests
**Files**: `test/widgets/dialogs/smart_plug_consumption_form_dialog_test.dart`, `test/screens/smart_plug_consumption_screen_test.dart`
1. Form tests: month/year picker renders, default to current month, validation, duplicate warning
2. Screen tests: new "März 2026 — 42.5 kWh" display format, edit/delete flow
3. Verify no interval type UI elements remain
