# valtra

Valtra ist eine App zur Erfassung und Analyse von Energie- und Wasserzählerständen (Strom, Gas, Wasser) über mehrere Haushalte hinweg, inklusive Verbrauchsstatistiken, Interpolation von Messwerten und Visualisierung der Nutzung bis auf Raum- und Geräteeebene.
